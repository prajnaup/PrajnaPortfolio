Projects

Boustrophedon based coverage path planning

Developed a coverage path planning algorithm using Boustrophedon Cellular Decomposition to enable mobile robots to systematically cover an environment while dynamically adjusting to obstacles. Implemented path re-evaluation techniques to optimize traversal time, ensuring seamless navigation in changing conditions. The algorithm efficiently decomposes the workspace into sub-regions, minimizing redundancy and improving coverage efficiency.

Design and Implementation of a Secure Company Network System

A new network service designed for the office building to ensure robust security to protect the network from internal and external threats incorporating VLANs, OSPF, HSRP, DMZ, and Cisco ASA firewalls. Configured inter-VLAN routing, EtherChannel, and dynamic/static IP allocation with DHCP. Established a robust architecture with firewall access controls, dual ISP redundancy, wireless LAN controllers, and VoIP integration. Ensured the network met all user requirements through testing and validation. Also ensured high availability, performance and scalability
